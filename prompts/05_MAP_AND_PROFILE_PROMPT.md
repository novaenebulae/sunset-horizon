# Prompt Cursor — Carte et profil

Améliore l'interface carte et graphique.

Objectifs :

1. Afficher le point observateur sur la carte.
2. Afficher la ligne d'azimut vers le coucher.
3. Afficher le point bloquant si disponible.
4. Afficher un graphique Recharts du profil altimétrique.
5. Mettre en évidence le point bloquant.
6. Ajouter états loading/error/success.

Contraintes :

- Composants typés.
- Pas de calcul lourd dans les composants.
- Le graphique doit rester lisible sur mobile.
