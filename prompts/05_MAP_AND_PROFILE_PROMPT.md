# Prompt Cursor — Carte et profil

Améliore l'interface carte et graphique.

Objectifs :

1. Ajouter une barre de recherche d'adresse afin de placer un point.
2. Afficher le point observateur sur la carte.
3. Afficher la ligne d'azimut vers le coucher.
4. Afficher le point bloquant si disponible.
5. Afficher un graphique Recharts du profil altimétrique.
6. Mettre en évidence le point bloquant.
7. Ajouter états loading/error/success.

Contraintes :

- Composants typés.
- Pas de calcul lourd dans les composants.
- Le graphique doit rester lisible sur mobile.
